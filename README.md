# Memorie_Test
이 프로젝트는 Python을 기반으로 개발한 치매 환자 상태 검사 및 간병 지원 프로그램입니다. datetime, random, time, os 모듈을 활용해 시간·날짜 기반 문항과 기억력 검사를 구성하였으며, MMSE-K 검사와 RAVLT 기억력 검사를 통해 사용자의 인지 상태를 간단히 점검할 수 있도록 구현하였습니다. 검사 결과는 Matplotlib을 이용해 점수 그래프로 시각화하였고, OpenAI API와 GPT-4o 모델을 연동한 간병 챗봇을 통해 치매 간병 관련 질문, 간병 팁 및 심리 상담형 응답을 제공하도록 설계하였습니다.
